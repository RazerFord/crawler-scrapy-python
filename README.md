# crawler-scrapy-python
Python crawler using the scrapy library

# Start project
`scrapy startproject bestsellers`

# Start shell
`scrapy shell`

# Netology courses
`https://netology.ru/backend/api/programs`

`scrapy runspider ./netology/netology/spiders/programs.py -o prog.csv`
